# Security Policy

KnobMixer uses Windows APIs for keyboard hooks, Raw Input, audio sessions, tray integration, and autostart.

If you report a security issue, include:

- Windows version;
- KnobMixer version or commit hash;
- reproduction steps;
- whether the app was launched as administrator;
- device type if the issue involves a keyboard, macro pad, or USB knob.

Do not open public issues for exploitable vulnerabilities. Contact the maintainer privately when possible.
